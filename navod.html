Spuštění

Do kódu HTML strány je potřeba přidat javascriptový soubor ze složky js/windows_manager.js a také stylový soubor css/style.css. 
<script type="text/javascript" src="./js/windows_manager.js"></script> 

<link href="./css/style.css" rel="stylesheet" type="text/css"/>
Tímto načtete příslušné funkce manažera, ale nespustíte jej. Pro spuštění je potřeba v kódu (před uzavíracím tagem </body>) vložit javascriptový kód: 
<script type="text/javascript>
  WindowsManager.run();
</script>

Použití

Manažer transformuje všechny elementy s css třídou .desktop na samostatné pracovní prostředí (tzv. workplace). V rámci tohoto elementu pak transformuje všechny elementy s třídou .window, ze kterých vytvoří jednotlivá okna plochy. S těmito okny je pak možné pohybovat, maximalizovat, měnit rozměry. Pro editaci vzhledu oken je potřeba upravit zmíněný soubor ./css/style.css. Existují předvytvořené třídy pro velikost okna:
  • .small
  • .medium
  • .large
Pro nadefinování nadpisu okna je možné použít html atribut title, který se po inicializaci oken nastaví do záhlaví okna a pak se atribut zruší pro další použití (V případě nedefinování se použije výchozí popisek okna "Empty Window").
